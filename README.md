# 🖼️ Img3D - تبدیل تصویر به مدل سه‌بعدی

**یک وب‌سرویس سبک و بدون وابستگی به OpenCV** که تصاویر دوبعدی را به فایل‌های OBJ سه‌بعدی تبدیل می‌کند.  
نسخه‌ی رایگان امکان تبدیل تصاویر به صورت آزمایشی را فراهم می‌کند و برای کیفیت حرفه‌ای، پرداخت از طریق **رمزینکس** (غیرمستقیم) انجام می‌شود.

## ✨ ویژگی‌ها
- **سبک و قابل اجرا روی Termux / سرورهای کم‌منبع** – بدون نیاز به OpenCV یا SciPy.
- **آپلود امن تصویر** با اعتبارسنجی نوع و حجم.
- **تولید فایل OBJ** قابل استفاده در نرم‌افزارهای سه‌بعدی (Blender، Maya و...).
- **پرداخت امن و غیرمستقیم** – بدون درگاه مستقیم یا افشای اطلاعات مالی؛ صرفاً تماس با پشتیبانی و دریافت لینک موقت رمزینکس.
- **رابط کاربری ساده و ریسپانسیو** با پشتیبانی از زبان فارسی.

## ⚙️ نصب و اجرا
۱. مخزن را کلون کنید:
```bash
git clone https://github.com/YOUR_USER/img3d.git
cd img3d
```

۲. وابستگی‌ها را نصب کنید (ترجیحاً در محیط مجازی):

```bash
pip install -r requirements.txt
```

۳. آیدی‌های پشتیبانی خود را تنظیم کنید
فایل public/backend/real_converter.py را باز کرده و مقادیر زیر را تغییر دهید:

```python
TELEGRAM_ID = "YourTelegramID"   # بدون @
EITA_ID = "YourEitaID"           # بدون @
```

(می‌توانید این مقادیر را از طریق متغیرهای محیطی نیز تنظیم کنید – بخش پایین را ببینید.)

۴. سرور را اجرا کنید:

```bash
python public/backend/real_converter.py
```

۵. در مرورگر به آدرس http://localhost:5000 بروید و یک تصویر آپلود کنید.

💳 پرداخت (نسخه حرفه‌ای)

این پروژه به‌گونه‌ای طراحی شده که هیچ درگاه پرداخت مستقیم وجود ندارد.
کاربران پس از آپلود تصویر، برای دریافت فایل با کیفیت بالا باید از طریق تلگرام یا ایتا با شما تماس بگیرند.
شما سپس یک لینک پرداخت موقت از رمزینکس (ریال یا تتر) برایشان ارسال می‌کنید. این روش هم امن است و هم نیازی به نمایش آدرس کیف پول در سایت ندارد.

🔐 تنظیم متغیرهای محیطی (پیشرفته)

اگر نمی‌خواهید آیدی‌های خود را مستقیماً در کد قرار دهید، می‌توانید از متغیرهای محیطی استفاده کنید.
ابتدا کد real_converter.py را به‌صورت زیر تغییر دهید (در خطوط مربوطه):

```python
import os
TELEGRAM_ID = os.environ.get("TELEGRAM_ID", "Default")
EITA_ID = os.environ.get("EITA_ID", "Default")
```

سپس قبل از اجرا، متغیرها را تنظیم کنید:

```bash
export TELEGRAM_ID="my_telegram_id"
export EITA_ID="my_eita_id"
python public/backend/real_converter.py
```
